# Security Policy

If you believe you've found a security vulnerability in `getmcpauth` or the mcpauth hosted service, please report it privately rather than opening a public issue.

**Report via [GitHub Security Advisories](https://github.com/yilmazali325/getmcpauth/security/advisories/new).**

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any relevant logs, requests, or code

We'll acknowledge reports as quickly as we can and keep you updated as we investigate.

See [getmcpauth.dev/security](https://getmcpauth.dev/security) for an account of current security practices.
